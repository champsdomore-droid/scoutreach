---

🎯 **What You'll Get From This**
Everything installed and ready so you can start building in the next lesson.

---

📝 **The Lesson**

You need three things. Most of you already have at least one.

**1. A Claude Account**

Go to [claude.ai](https://claude.ai) and sign up. The free tier works to get started. If you want to follow along with everything in this community, here is how the plans break down:

🟢 **Free** — You can chat with Claude in the browser. Good enough to learn prompting and try the folder structure by pasting content in manually.

🟡 **Pro ($20/mo)** — More messages, access to the strongest models, file uploads. This is where most members sit. If you are going to pick one paid plan, this is it.

🔵 **Claude Code (comes with Pro/Max)** — This is Claude running in your terminal. You point it at your files and it reads, writes, and edits them directly. Most of what we teach uses this. If you watched the folder architecture video, Claude Code is how that system actually runs.

You do not need a paid plan to get value from this community. You do need an account.

**2. A Code Editor**

You need somewhere to keep your files. Two good free options:

🟢 **VS Code** — The standard. Free. Works on Mac, Windows, Linux. If you have never used a code editor before, start here.

🟢 **Cursor** — Built on VS Code but with AI features baked in. Also free to start. Some members prefer this.

Pick one. Install it. You do not need to know how to code. You need a place to organize files and folders. That is what we use it for.

**3. Node.js (Only if using Claude Code)**

If you plan to use Claude Code (the terminal tool), you need Node.js installed. It takes two minutes.

Go to [nodejs.org](https://nodejs.org), download the LTS version, and run the installer. If you are not planning to use Claude Code yet, skip this and come back when you are ready.
