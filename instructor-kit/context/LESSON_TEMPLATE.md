# Lesson Page Template

Use this structure for every lesson page in Skool. Every section has a job. Skip sections that do not apply (not every lesson has a video or downloadable resources), but keep the order and emoji markers consistent so members always know where they are.

---

## The Template

```
# LESSON [X.X]: [Title]

---

🎯 **What You'll Get From This**
[One sentence. Write it as an outcome, not a topic.]
[Good: "Everything installed and ready so you can start building in the next lesson."]
[Bad: "An overview of setup steps."]

---

▶️ **Video**
[Embed if this lesson has one. Skip this section entirely if not.]
[Add a one-line note: "The written version below covers the same ground. Watch, read, or both."]

---

📝 **The Lesson**
[This is the core content. Written to stand on its own even without the video.]
[Short paragraphs. Bold key terms on first use. Headings to break up long lessons.]
[If this lesson has a video companion, follow the video's flow but add a]
["what this means for you" bridge at the end connecting it back to practical work.]

---

📥 **Resources**
[Only if this lesson has downloadable files, templates, reference cards, or tool links.]
[One line per resource with a brief label. Link directly to the thing.]

---

🔗 **Go Deeper**
[2-4 links. Each with a one-line reason to click. Four types:]
[1. Prerequisite: "If you skipped 1.1, go do that first."]
[2. Expansion: "We go deeper into this in [Course Name] (Level 2)."]
[3. Related lesson: "This connects to what we cover in [X.X: Title]."]
[4. External: "Anthropic's docs on this: [link]"]

---

💬 **Check In**
[Link to the dedicated discussion post for this lesson.]
"[Specific action prompt. Ask them to DO or SHARE something.]"

---

⏭️ **Next Up: [Lesson Title]**
[One sentence teaser. Link to next lesson.]
```

---

## Numbering

Lessons use section.lesson format: 1.1, 1.2, 2.1, 2.2, etc. The section number matches the Skool course section. The lesson number is sequential within that section.

---

## Placeholder format

When content is not ready, use placeholders. They keep the structure honest and become a visible task list.

```
[📌 INSTRUCTOR: Description of what needs to be added]
[🎨 IMAGE PROMPT: "Ready-to-paste prompt for generating a graphic"]
[LINK TO X.X] - Fill in once the lesson exists in Skool
[VIDEO: Recording TBD - brief description of what it should cover]
[RESOURCE: Name - not yet created]
```

Never skip a section because the content is not ready. The placeholder is better than the gap.
